# Audio Files for Kitchen Display

## Required Audio File

Place a sound file named `new-order.mp3` in this directory for kitchen order notifications.

### Recommended Sound:
- **Format:** MP3
- **Duration:** 1-3 seconds
- **Volume:** Medium (not too loud)
- **Type:** Pleasant notification sound (e.g., bell, chime, ding)

### Free Sound Resources:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/

### Example Search Terms:
- "notification bell"
- "order ready"
- "kitchen bell"
- "service bell"

### Note:
The kitchen display will work without this file, but audio notifications won't play. The system will silently fail if the file is missing.
